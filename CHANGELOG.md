## [1.0.0] - 2026-07-01

### 🎉 الإصدار الأول

الإصدار الأول من إضافة **Fill Attachment Fields** لتصفح Google Chrome — أداة احترافية لتسريع عملية ملء حقول المرفقات في ووردبريس من عنوان المنتج، تعمل على أي موقع ووردبريس بدون إعدادات مسبقة.

---

### ✨ ميزات جديدة

#### 🔧 الميزات الأساسية

- **دعم أي موقع ووردبريس**
  - تعمل تلقائيًا على أي موقع ووردبريس — جذر النطاق أو مجلد فرعي
  - لا حاجة لإعدادات مسبقة — تعمل فور التثبيت
  - كشف تلقائي لصفحات تحرير المنتجات (`post.php` / `post-new.php`)

- **زر "املأ من عنوان المنتج"**
  - يظهر في أعلى لوحة تفاصيل كل مرفق داخل مكتبة الوسائط
  - تصميم متوافق مع واجهة ووردبريس
  - نص قابل للتخصيص من الإعدادات

- **ملء الحقول الأربعة**
  | الحقل | الوصف | الأهمية |
  |------|-------|---------|
  | النص البديل (Alt Text) | نص بديل للصورة — مهم لـ SEO وإمكانية الوصول | ⭐⭐⭐ |
  | العنوان (Title) | عنوان المرفق في ووردبريس | ⭐⭐ |
  | كلمات توضيحية (Caption) | النص الظاهر أسفل الصورة عند العرض | ⭐ |
  | الوصف (Description) | وصف تفصيلي للمرفق | ⭐ |

- **مراقبة DOM تلقائية**
  - `MutationObserver` مع `debounce` (200ms) للأداء الأمثل
  - اكتشاف لوحات المرفقات الجديدة فور ظهورها
  - إضافة الزر تلقائيًا بدون تحديث الصفحة

#### 🖥️ النافذة المنبثقة (Popup)

- **تصميم احترافي عصري**
  - تصميم بطاقات مع تدرجات لونية وظلال
  - دعم RTL كامل — واجهة عربية أنيقة
  - أيقونات SVG مدمجة لكل عنصر

- **مكونات النافذة**
  - بطاقة الحالة — تفعيل/تعطيل الإضافة مع مؤشر بصري
  - شبكة إحصائيات سريعة — عمليات الملء · الصور · المواقع
  - أزرار اختيار الحقول — تصميم chip مع أيقونات
  - خيارات متقدمة — الملء التلقائي + نص الزر المخصص
  - بطاقة الاختصار — تذكير بـ `Alt+Shift+F`
  - زر الإعدادات الكاملة — رابط لصفحة Options
  - تذييل — رقم الإصدار + روابط التواصل (واتساب · بريد · GitHub)

- **تبديل المظهر**
  - زر تبديل في الهيدر بين الوضع النهاري والليلي
  - ثلاثة أوضاع: تلقائي (حسب النظام) / نهاري / ليلي
  - انتقال سلس بين الأوضاع

#### ⚙️ صفحة الإعدادات (Options Page)

- **نظام تبويبات مع شريط جانبي**
  - شريط جانبي أنيق مع أيقونات لكل تبويب
  - تنقل سلس مع تأثيرات انتقال
  - رابط GitHub في أسفل الشريط

- **تبويب عام**
  - تفعيل/تعطيل الإضافة بالكامل
  - تفعيل/تعطيل الملء التلقائي عند الرفع
  - اختيار المظهر: تلقائي / نهاري / ليلي (بأزرار اختيار مرئية)

- **تبويب الحقول**
  - التحكم في كل حقل بشكل منفصل
  - شرح تفصيلي لكل حقل وأهميته
  - أزرار toggle أنيقة

- **تبويب المواقع**
  - إضافة مواقع إلى القائمة المحظورة
  - حذف مواقع من القائمة
  - حالة "فارغ" عند عدم وجود مواقع محظورة

- **تبويب الإحصائيات**
  - بطاقات إحصائيات ملونة (عمليات الملء · الصور · المواقع)
  - آخر وقت استخدام
  - ترتيب المواقع الأكثر استخدامًا
  - زر تصفير الإحصائيات مع تأكيد

- **تبويب حول**
  - معلومات الإضافة والإصدار
  - قائمة المميزات الرئيسية بأيقونات
  - بطاقة بيانات المطوّر مع روابط تواصل
  - أزرار GitHub والإبلاغ عن مشكلة

#### ⌨️ اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| `Alt + Shift + F` | ملء حقول المرفق الحالي من عنوان المنتج |
| `Alt + Shift + P` | فتح النافذة المنبثقة للإعدادات السريعة |

> يمكن تعديل الاختصارات من `chrome://extensions/shortcuts`

#### 📊 نظام الإحصائيات

- **عدادات رئيسية**
  - عدد عمليات الملء الإجمالية
  - عدد الصور المملوءة
  - عدد المواقع المختلفة

- **تتبع متقدم**
  - آخر وقت استخدام
  - ترتيب المواقع الأكثر استخدامًا (مع عدد العمليات لكل موقع)
  - شارة على أيقونة الإضافة تعرض عدد عمليات الملء

- **إدارة**
  - تصفير الإحصائيات مع تأكيد
  - حفظ محلي في `chrome.storage.local`

#### 🎨 التصميم والواجهة

- **متغيرات CSS** — نظام ألوان موحد مع دعم الوضع الليلي
- **خط Cairo** — خط عربي احترافي
- **تصميم متجاوب** — يعمل على جميع أحجام الشاشات
- **تأثيرات حركية** — انتقالات سلسة وتأثيرات hover
- **أيقونات SVG** — مدمجة وخفيفة بدون مكتبات خارجية

#### 🔒 الخصوصية والأمان

- **لا جمع بيانات** — الإضافة لا تجمع أي بيانات شخصية
- **إعدادات محلية** — `chrome.storage.sync` (تتزامن مع حساب Chrome فقط)
- **إحصائيات محلية** — `chrome.storage.local` (لا تُرسل لأي خادم)
- **لا طلبات خارجية** — لا توجد أي اتصالات شبكة خارجية

### 🏗️ تقني

| المكون | التفاصيل |
|--------|----------|
| **Manifest** | V3 — متوافق مع أحدث سياسات Chrome |
| **Service Worker** | `background.js` — إدارة الإحصائيات والاختصارات وتحديث الشارة |
| **Content Script** | `content.js` — المنطق الأساسي: قراءة العنوان، إضافة الزر، ملء الحقول |
| **Storage** | `chrome.storage.sync` للإعدادات + `chrome.storage.local` للإحصائيات |
| **i18n** | دعم تعدد اللغات — عربي (`ar`) / إنجليزي (`en`) |
| **أيقونات** | 16×16 · 48×48 · 128×128 · 512×512 — تصميم احترافي |
| **DOM Observer** | `MutationObserver` مع `debounce` (200ms) |
| **أحداث ووردبريس** | إطلاق `input` + `change` + `blur` لضمان الحفظ التلقائي |
